## FORMAT
### OBSIDIAN FORMAT


#### Line Format
- STATUS - Single character representing current status
- NAME - The [[AT]] link for the person
- DATE - Optional date by which action should be taken
- NOTES - Explanation of current state or goals for the relationship

### STATUS
- !	Urgently required communications.  !! and !!!
- A	Contact ASAP.  There is some specific timing-related reason to connect in a timely fashion
- N	Contact Next / Now.  (not waiting on anything)
- S	Contact soon
