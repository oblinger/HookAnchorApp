-   **ROBUST EXECUTION**. All functions in both apps execute robustly. This requires, native recording [R1.10](https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45088809 "https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45088809"), background uploading [R1.8](https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/44040323 "/wiki/spaces/~6044e56c03ceda006979f4f5/pages/44040323"), handling low battery [R1.9](https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45350952 "/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45350952"), do not disturb mode [R1.7,](https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45449285 "/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45449285") and clear all known bugs in upload [R1.15](https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45219881 "/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45219881").  
    Robust execution requires managing poor/no wifi including: account persistence [R1.6](https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/44269720 "/wiki/spaces/~6044e56c03ceda006979f4f5/pages/44269720"), and game fetching [R1.5](https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45350945 "/wiki/spaces/~6044e56c03ceda006979f4f5/pages/45350945").
    
-   **SINGLE CAMERA RECORDING**. The ability to record games with a single camera and a Pivo using manual control [R1.3](https://software-engineering-team.atlassian.net/wiki/spaces/~6044e56c03ceda006979f4f5/pages/43221067 "/wiki/spaces/~6044e56c03ceda006979f4f5/pages/43221067").
