FROM


  Punctuation Commands

  - "period" or "full stop" (UK) → .
  - "comma" → ,
  - "question mark" → ?
  - "exclamation point" → !
  - "colon" → :
  - "semicolon" → ;
  - "apostrophe" → '
  - "hyphen" or "dash" → -
  - "ellipsis" → ...
  - "ampersand" → &
  - "asterisk" → *
  - "at sign" → @
  - "hashtag" or "pound sign" → #

  Quotation Marks

  - "open quote" → "
  - "close quote" → "
  - "quote" and "unquote" → for paired quotes

  Parentheses & Brackets

  - "open paren" → (
  - "close paren" → )
  - "open bracket" → [
  - "close bracket" → ]

  Line & Paragraph Formatting

  - "new line" → creates a line break (↵)
  - "new paragraph" → creates a paragraph break (↵↵)

  Capitalization

  - "caps on" / "caps off" → toggle capitalization
  - "all caps" → makes the next word all uppercase
  - "no caps" → makes the next word lowercase

  Numbers

  - "numeral" before a number → enters digit instead of word
    - Example: "numeral four" → 4 (not "four")
