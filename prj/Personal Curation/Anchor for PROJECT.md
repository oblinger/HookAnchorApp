
# Project Anchors
The most common stream types are project streams.  Each anchor project in these streams have a number of optional aspects listed here:



## Anchor File Structure
- An anchor is a folder containing a Markdown file with the same name as the folder.  So:
  `.../My Anchor Example Project/My Anchor Example Project.md` would be the markdown summarizing this anchor folder.

### Acronym Naming
- Commonly accessed anchors will have an acronym for quick access, usually 3 or 4 letters long.
