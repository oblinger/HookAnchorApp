## About This Template

This is an example anchor folder structure. Copy this folder and rename:
1. The folder itself (e.g., `My Project/`)
2. The redirect markdown (to `My Project.md` containing `See [[{NAME}]]`)
3. The `{NAME} Docs/` folder
4. The `{NAME}.md` file inside Docs
5. All `ATV` prefixes to your `{NAME}`

Then update the link table to point to your actual resources.
