## About This Template

This is an example anchor folder structure. Copy this folder and rename:
1. The folder itself
2. The folder-named markdown (to your new folder name)
3. The TLC.md file (to your chosen TLC)
4. All `ATV` prefixes to your TLC

Then update the link table to point to your actual resources.
