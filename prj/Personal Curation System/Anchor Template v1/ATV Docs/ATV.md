## About This Template

This is an example anchor folder structure. Copy this folder and rename:
1. The folder itself (e.g., `My Project/`)
2. The redirect markdown (to `My Project.md` containing `See [[TLC]]`)
3. The `{TLC} Docs/` folder (to match your TLC)
4. The `{TLC}.md` file inside Docs (to your chosen TLC)
5. All `ATV` prefixes to your TLC

Then update the link table to point to your actual resources.
