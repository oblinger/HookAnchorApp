# ANCHOR FOLDER

The most common stream types are streams of anchors.  Each anchor usually associates with a folder.  Here are the aspects that may associate with an anchor folder.

## Anchor Folder File Structure
An ***`"anchor"`*** is a folder that contains a Markdown file with the same name as the folder.  So:
  `.../My Anchor Example Project/My Anchor Example Project.md`  indicates a markdown folder.


### Acronym Naming
- Commonly accessed anchors will have an acronym for quick access, usually 3 or 4 letters long.
