=[[prj]] 

Works with the File This online service to present new financial statements for weekly viewing and monitoring.

1. **File This** online service 					--  Copies statements from each financial institution
2. **Google Drive** for oblinger@gmail.com 		--  Cloud service used to store these statements
3.  /Users/oblinger/My Drive/FileThis			--  Local folder location where statements are synced from the "Google Drive" service
4.  /Users/ob/bin/filethis						--  Script that computes update deltas on this synced folder for previewing
5.  /Users/oblinger/.filethis_index.json		--  File location of timestamp data used by this script
6.  /Users/oblinger/.filethis_viewfolder		--  Folder used for viewing new statement files
