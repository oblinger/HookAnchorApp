# App.Motivation --

    GOAL:  A semi-automated employee skill-profiling system 
     
    GOAL:  Document an organization's skill assets w/o significantly taxing 
           those assets. 
     
    SOLUTION: 
    - Use e-communications to infer employee skills 
    - Allow user to confirm profile entries. 
      - Ensures privacy. 
      - Improves accuracy and completeness. 
    - Adapt skill inference based on user changes. 
     
    BENEFITS OF AUTOMATION: 
    - Impetus to complete profile 
    - All employees have skill info (even if they ignore system) 
    - Continuously updated 
    - Saves large amt of time over entire organization 
    - Results in a more complete & accurate hierarchy 
     
    BENEFITS OF EMAIL: 
    - Ubiquitous 
    - Rich source of info 
    - Up to date 
    - Low/no effort per employee to aquire data
