# Docs.CREATE\_NOTE\_CMD --


    The CREATION menu is used to create new notes.  (See <<>> to select how to add)

    The first character of each entry can be used to select that item.
    See <<>> for [re-]configuration.

    ACTIVITY -- General activity (not within some project)
    JOURNAL  -- Entry in the General journal (not within some project)
    MEETING  -- General meeting entry (not associated to a project)
    PROJECT  -- New toplevel project (not within another project)
    TOPIC    -- General toplevel topic (topics do not associate with a folder like projects do)
