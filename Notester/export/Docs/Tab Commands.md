# Docs.Tab Commands --


              --- Notester commands activated by TAB or CTRL and the letter ---
    <<>>
    | A <<>>        ATTACH a file        | N <<>>       changes note NAME    |
    | B <<>>          browse BACK to prev  | O                   insert datestamp     |
    | C                                          | P <<>>       changes note PARENT  |
    | D <<>>        DELETES current note | Q <<>>    QUICK create menu    |
    | E <<>>       MS Folder EXPLORER   | R                   Change list VIEW     |
    | F                                          | S <<>>      SEARCH menu          |
    | G <<>>          GOTO a note by name  | T <<>>      THROW note to list   |
    | H <<>>          shows the HELP       | U <<>>          go UP to parent note |
    | I <<>>       ICONIFIES notester   | V <<>>                             |
    | J <<>>          JUMPS to freq notes  | W <<>>      WITHIN project create|
    | K                                          | X                                        |
    | L <<>>     Goes to LINK in note | Y                                        |
    | M <<>>                              | Z                                        |
    <<>>

    ALT    Name, Parent, Up, Down
    ALT2   Delete, Disown, Move, Throw, QuickCreate, Within
