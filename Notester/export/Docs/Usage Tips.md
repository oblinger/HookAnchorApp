# Docs.Usage Tips --




    Starting lists

    IN BOX  (Things can be put here if there is not time to sort them immediately)

    SMALLER TASKS  (One-Item)
    - Todo       Quad 1:          Urgent,       Important       What I am working on now
    - Oppertune  Quad 2:  Not-Too-Urgent,       Important       Important tasks w/o deadline (mub must be done 'soon')
    - B-List     Quad 3:  Non-    Urgent,  Some-Importance      Somewhat important tasks that can be done later
    - Joy        Quad 4:  Non-    Urgent,   Non-Important       Fun stuff, mindless stuff

    LARGER TASKS (take more time, involve many sub-parts, require planning and scheduling)
    - Week       Tasks active this week (plus goals for week)
    - Prime      Prime tasks currently active
    - Monitor    



    - Lists      Place to stick informational lists and pages
