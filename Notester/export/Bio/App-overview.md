# Bio.App-overview -- Overview of the goals of my bio app wrt my thesis

    GOAL:  Perform a cross-validated experiment that uses bio. knowledge 
           to improve performance. 
     
    *   Possible performance tasks: 
        1) Fold class prediction. 2) Alignment generation. 
    *   Possible Evaluation Metrics: 
        + Leave-1-out accuracy.  + Avg. Z-score.  + Avg. min-rms. 
        + Distance from correct structural alignment. 
        + Structural Accuracy on one fold class (Tim Barrels)
