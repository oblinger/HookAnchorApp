# NH.Navigation --


    A central goal of Notester is facilitate *VERY* *QUICK* navigation
    - To view a specified 
    - To 'throw' new information to a specified target
    - To execute some notester or filesystem command


    METHODS
    - Goto note by name.  
      (can use aliases to provide abbreviations for common notes)
    - Traverse a prefix-character tree.
      - this tree should be defined by note child relation.  note titles should specify display text.  redundant prefix chars should cause numbers to be used.
