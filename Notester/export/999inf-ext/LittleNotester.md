# 999inf-ext.LittleNotester --

    Idea: A simple HTML based hypertext notes app. 
    Features: 
     
    Operations: 
    - Snarf selected text w. keystroke 
      ENTER to paste it 
      TYPE to name it; TAB to title it 
    - Create child 
    - Rename node 
    -  
     
     
     
    - A node is the basic building block 
    - An atomic node is a packet of text. 
    - A node may be a list of nodes 
    - A node may be named 
     
     
     
    Files can be stored in .zip structure 
    HTML file name = node name 
    HTML dir name = node name for nodes with sub-nodes 
    a hrefs are node links
