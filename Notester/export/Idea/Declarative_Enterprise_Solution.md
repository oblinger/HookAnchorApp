# Idea.Declarative\_Enterprise\_Solution --



    FEATURES
     * All parameters of all components are functionally <<>> declaratively specified
     * Group of components can be encapsulated into a component
     * Component/parameter algebra
     * Component versioning

    TYPES OF COMPONENTS
     * Msg queue
     * Web server
     * Machine Execution Environment
       * Process Execution Environment
       * Compile pipe

    TYPE OF DATUM
     * Structured Record
       * User Session
       * Transaction



    COMPONENT ALGEBRA
     * Instantiate
     * Snapshot: 

    PARAMETER ALGEBRA
     * Parameter can be derived functionally, declaratively, set as constant, or incrementally set/updated
     * Parameter updates can be sychronized manually, can be automatic.
