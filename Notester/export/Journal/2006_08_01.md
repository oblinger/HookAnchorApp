    Sections of the Ladder API
    --------------------------
    - Predicate Calculus
        int, float, string, symbol (case insensitive), keyword
        predicate/function

    - Package mechanism
      - Package foo <<>>

    DefSyntax ListOfFoo extends List <<>>


    ---------------------
    Packages are just lists of typed elements, so 

    Package zeeble of Rule;
    Rule( if: =($x $y) then: doit() )


    would expand to:

    DefSyntax PackageOfRule extends Package <<>>

    PackageOfRule( elenents: Rule( if: =($x $y) then: doit() ) )
