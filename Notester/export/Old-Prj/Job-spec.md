    Java GUI Programmer  - Applied Learning Sciences Group<<<>>> 
      
    Job Description:<<<>>>  
    The Applied Learning Sciences Group at the T J Watson Research 
    Center investigates and develops new technologies for use in 
    educational and training applications. We are working as part of a 
    larger team to integrate a number of sources of information into a 
    novel visual framework. 
     
    We are seeking one talented Java GUI programmer for this effort. 
    This work involves software design, development, and testing.  
      
    Skills:<<<>>> 
     
    Four or more years of experience in programming required (C/C++/Java) 
    and at least two years programming significant Java GUI applications. 
    Knowledge of basic Web technologies (Netscape Navigator, Microsoft 
    Explorer, HTTP, CGI), object-oriented programming, graphical user 
    interfaces, and SQL relational databases is required. Previous project 
    experience programming server-side code to handle news groups or 
    discussion forums and/or using text analysis tools (e.g., IBM 
    Intelligent Miner for Text) and techniques (e.g., feature extraction) 
    is highly desirable. Knowledge of Java servlets, Microsoft Windows NT, 
    Visual C++, Visual Age for Java, Domino, and DB2 is also 
    desirable. Experience in the computer industry or education and 
    training industry and/or experience as a Webmaster for a ".com" 
    company or corporate Intranet site is a plus. 
     
    Educational level:<<<>>> 
    BS in CS required.
