# 999icl.Parts --

    An Environment is a namespace.   
    Each namespace is a set of attribute value pairs. 
    Each value is an object. 
    Each object can be a namespace, set, vector of object or may be base object 
    type. 
     
     
    The dependance hierarchy is a partial ordering over the combined environments. 
    (Plus combination parameters) 
     
    COMPOSITIONAL PRIMATIVES 
    - Set inclusion/exclusion statements 
    - Vector ordering constraints 
    - Attribute value setting 
    - Attribute value updating (read; compute; update) 
    - Numeric
