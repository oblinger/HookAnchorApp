    BASIC PROCESS
    (1) COLLECT *everything* and place it "in" your In Box.
    (2) PROCESS each item in In Box:
        Is it actionable?
        NO:    
               May you want to take action on it at some point?    YES:  Add it to tickle/someday/maybe list
               Do/will you need it as reference?                   YES:  File it
                                                                   NO:   Trash it
        YES:   
               Does it take less than 2 minutes?                   YES:  DO-IT
               Are you the most appropriate person to do it?       NO:   DELEGATE-IT
               Does it require planning?                           YES:  PLAN-IT, add first step(s) to Next Actions list
                                                                         DEFER-IT, determine next action(s) and add it to appropriate context list
    (3) EXECUTE
        Select actions for current context as appropriate






    [[BIG POINTS]]

    GENERAL FILING SYSTEM
    - Alphabetic by name
    - File single entries
    - Purge occasionally

















    PRIMARY TIPS
    - Have defined workable workspace
    - Have general filing system
    - Have lists
    - Do weekly review
