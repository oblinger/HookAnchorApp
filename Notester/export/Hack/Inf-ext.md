    > PROPOSAL 2/13/01 
    * Create phone log entry that is also a NODE 
    * Name is date gensym  YYMMDDX 
    * GUI: 
      - User enters snarf command 
      - System prompts for title (and other title commands) 
      - Page is created and displayed for user  
      - User types ^X^S ALT-TAB to return to work 
    * back-tick I I ? (snarf command) 
      I insert   add obj's content to body 
      L link     *COPY* obj to created file; add file link to body 
      R ref      add url reference to body; no attach file created 
      CA attach  clipboard attach 
      CI insert  clipboard insert 
      A attach   save attachment to file; add link to body 
    * Title commands 
             done entering commands 
      =          specify name of new node 
      -          don't create new node, just append to old node 
    * The I I [ALI] commands work in the following contexts 
      - With text selected 
      - Reading a notes note 
      - viewing a web page 
     
     
      STANDARD 
      - A title line 
      - A unique creation stamp: YMDDL   (used as file prefix too) 
     
     
     
      OPTIONAL STUFF 
      - Body text 
      - File system object (that 'start' operates on) 
      - A URL 
      - A unique name (and storage location) 
      - A seperate directory 
     
     
      COMMANDS 
      - Create new 
      - Edit (create from previous) 
      - Move 
      - Attach
