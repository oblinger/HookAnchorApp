

# Uploading UX

1. Download & run app.
    User enters a nickname for this device
2. Press record; record game
3. Press stop > system provides menu to select the target folder to save game
   User can click the trashcan to delete the video


--> Upload status is highlighted with:
       file count (1 of 3) 
       percent uploaded   57%
       and as a progress bar until upload is complete



- User can change nickname by clicking on it.
- User can change target folder by clicking on it.
- User can press "record" to record a new video anytime and press stop to send it.
	- By default it will go to the target set
