


### Spec Type
type = spec of the structure of a collection

u(h:typespec, int: unitspec, named: unitspec, 
	meta: unitspec, has: us, sat: predicate)

domain(typespec)
range(typespec)


### Spec Interface

interface = spec of the function of a collection

fn(int: class, named: class, meta: class, ->: class)

op(self: class, ...)

return(ispec)

### Spec Class
unitspec = class = spec of structure and function
