| ANABOLIC PATHWAYS        | What it builds                           | Energy Input             |
| ------------------------ | ---------------------------------------- | ------------------------ |
| **Gluconeogenesis**      | Glucose from non-carb sources            | ATP, GTP                 |
| **Fatty acid synthesis** | Fatty acids from acetyl-CoA              | ATP, NADPH               |
| **Protein synthesis**    | Proteins from amino acids                | GTP (during translation) |
| **Nucleotide synthesis** | DNA and RNA bases from simple precursors | ATP                      |
| **Glycogenesis**         | Glycogen from glucose                    | UTP                      |

| CATABOLIC PATHWAYS                      | What it breaks down      | Key Products                               |
| --------------------------------------- | ------------------------ | ------------------------------------------ |
| **Glycolysis**                          | Glucose → Pyruvate       | 2 ATP, 2 NADH                              |
| **β-Oxidation**                         | Fatty acids → Acetyl-CoA | NADH, FADH₂, Acetyl-CoA                    |
| **Proteolysis**                         | Proteins → Amino acids   | Amino acids (enter various paths)          |
| **Citric Acid Cycle (TCA)**             | Acetyl-CoA → CO₂         | 3 NADH, 1 FADH₂, 1 GTP                     |
| **Pentose Phosphate Pathway (partial)** | Glucose-6-phosphate      | NADPH, ribose-5-phosphate (partial energy) |


| ENERGY TRANSPORT            | Main Pathway / Cycle          | Energy Form Handled                           | Key Notes                                 |
| --------------------------- | ----------------------------- | --------------------------------------------- | ----------------------------------------- |
| **Carbohydrate metabolism** | **Glycolysis**                | Glucose → ATP, NADH                           | Fast ATP, anaerobic or aerobic            |
|                             | **Gluconeogenesis**           | Uses ATP to make glucose                      | During fasting                            |
|                             | **Glycogenesis**              | Stores glucose as glycogen                    | Needs UTP                                 |
|                             | **Glycogenolysis**            | Releases glucose from glycogen                | Quick access to glucose                   |
| **Fat metabolism**          | **β-Oxidation**               | Fatty acids → Acetyl-CoA, NADH, FADH₂         | Huge ATP output, slow start               |
|                             | **Fatty acid synthesis**      | Acetyl-CoA + NADPH → fatty acids              | Consumes lots of energy                   |
| **Amino acid metabolism**   | **Proteolysis**               | Proteins → amino acids → energy intermediates | Emergency energy source                   |
|                             | **Urea Cycle**                | Detoxifies ammonia                            | Costs ATP                                 |
| **Energy generation**       | **Citric Acid Cycle (TCA)**   | Acetyl-CoA → NADH, FADH₂, GTP                 | Central hub of energy harvesting          |
|                             | **Electron Transport Chain**  | NADH, FADH₂ → Proton gradient → ATP           | Major ATP factory                         |
| **Alternative pathways**    | **Pentose Phosphate Pathway** | NADPH for biosynthesis                        | Not for ATP directly, but critical        |
|                             | **Cori Cycle**                | Lactate → Glucose (between muscle and liver)  | Recycles lactate after anaerobic exercise |
|                             | **Alanine Cycle**             | Alanine → Glucose (muscle-liver shuttle)      | Alternative to Cori for amino acids       |
