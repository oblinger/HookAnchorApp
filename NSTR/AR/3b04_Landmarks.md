# AR.3b04\_Landmarks --



    LANDMARK --  A landmark is a Logical point within a procedure

    LANDMARK RECOGNIZABILITY -- This is a measure of how easily one
    can recognize that a procedure execution is current at that landmark

    LANDMARK LEVEL -- Within a hierarchical procedure, this is a measure
    how high within the hierarchy the landmark is.

    LANDMARK COMMONALITY -- Measure of the execution path fan-in and 
    fan-out of a landmark node within a procedure.


    USES OF LANDMARK --
      Assist in segmentation
      Assist in retrieval
