# Remember\_Every\_Name\_Everytime --


    FACE
      Focus       -- Get "in zone" before introduction.  *FOCUS* on them when they say name
      Ask         -- Ask to ensure you have the name correct and to reinforce
      Comment     -- Comment verbally or internally on name.   "always think, XXXX *LIKE* YYYY"    Categorize, Cross-reference, Connect
      Employ      -- Employ name towards end of conversation, at ends of sentences

    NAME
      Nominate    -- Nominate some facial feature
      Articulate  -- Articulate some details about that feature
      Morph       -- Morph their name into a physical object
      Entwine     -- Entwine the morph with the nominated feature
