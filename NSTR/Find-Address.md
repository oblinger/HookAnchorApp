# Find-Address --

    UNIX FIND-ADDRESS: Finding remote sites on the internet 
     
     
    To resolve an internet address you must: 
    1) Find a nameserver that knows your destination machine/area 
    2) Find your destination hosts name 
    3) Find the destination user on you host 
     
    Helpful commands: 
     
     whois     Returns information about a 'domain' area. 
     host      Finds info about first host asking second. 
     nslookup  Ask server to resolve address. 
     
     archie    Archie program searching utility
