# Habits.Memory\_Triggers --


    === TRIGGERS ===

    - MORNING TRIGGER
      - Arrange items for pockets: iphone,wallet,keys,hankerchief
      - Arrange bookbag, check home pocket
      - Check Calendar/Home list
      - *THEN* put stuff in pockets once all is done

    - WHEN TRAVELLING
      - Wallet etc. *always* goes into back top pocket of backpack


    === MOTIVATION ===
    - The goal is to effortlessly remember to do things with high reliability.
    - To do this we use physical cues to remember to check for certain actions.
