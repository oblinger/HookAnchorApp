# Docs.VIEW\_CMD --


    The VIEW command presents a menu of different layouts for Notester.  Wait for user to press key to select new view.

    Predefined views include:
     START    -- This is the main notester view.  It displays the daily waterfall (see <<>>)
               as well as context lists and planning waterfall lists that are viewed on a daily basis.
     HOME     -- View used at home, it displays personal/home lists
     WEEKLY   -- Views the lists in the START view, plus a single pane where lists scanned once a week are listed.
     PLANNING -- View used for longer term planning.

    See <<>> to [re-]configure new view.
