### CAPS LOCK MENU
- KARABINER maps caps alone to Opt-Cmd-F12
- KEYBOARD MAESTRO execute "global command" macro for this keystroke
	- Starts and raises Obsidian
	- Presses  Shift-Cmd-O
- Triggers the special switcher called "Dan's Switcher" as configured in the "Another Quick Switcher" plugin


### CAPS LOCK CONTROL KEY 
- KARABINER maps caps lock with another key as the hyper modifier (Shift-Control-Option-Command)
- SEE [[KBD Shortcuts]] for list of defined caps commands (some are defined in keyboard maestro with prefix "CAPS")


### TAB-KEY
- KARABINER maps tab alone into a 'tab' key
- KARABINER maps tab with another key into a (Control-Command) modifier combination
- SEE [[KBD Shortcuts]] for list of such tab commands (some are also defined in keyboard maestro with prefix 'TAB')



### KEY-REMAPPING
#### CAPS-LOCK
- CAPS operates as HYPER (Shift-Control-Option-Command)
- CAPS alone generates Opt-Cmd-F12 which runs my Keyboard Maestro "global command" script

#### TAB-KEY
- TAB-ALONE --> Generates a 'TAB' key     (via Karabiner Rule)
- TAB-MODIFIER --> Same as CTRL-CMD
- TAB-Q --> CTRL-CMD-Q  (use System Preferences -> Keyboard -> Keyboard Shortcuts -> App Shortcuts -> All Applications
	Press "+"  then add 'Lock Screen'  Shift-Ctrl-Cmd-Q    (this changes the default lock screen keyboard shortcut)
- TAB-Q --> CTRL-CMD-Q --> Keyboard Maestro Macro to insert a tab key
